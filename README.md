# ABOUT

Simple Vue app for a static website.

# SETUP

## Requirements and installation

`Node.js v18.3.0`
`npm@8.11.0`

Can be installed from nodejs.org

`@vue/cli 5.0.4`

`npm install -g @vue/cli`

## Serving locally

`cd` into `./data_analytics_website`

`npm run serve`
