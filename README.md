# ABOUT

Simple Vue app to display my own portfolio.

# SETUP

## Requirements and installation

`Node.js v18.3.0`
`npm@8.11.0`

Node/npm can be installed from nodejs.org

`@vue/cli 5.0.4`

`npm install -g @vue/cli`

## Serving locally

`cd` into `./portfolio`

`npm run serve`
